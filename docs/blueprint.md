# **App Name**: ForgeHive

## Core Features:

- Curated Tool Browsing: Browse a public list of curated open-source tools with name, description, category, and external link.
- Search and Filter: Keyword-based search and category filters to find the right tools quickly.
- Tool Detail Page: See the full description and a direct external link for each tool.
- Authentication: Email/password authentication managed by Supabase Auth with persistent sessions.
- Save to Favorites: Logged-in users can save tools to their favorites list, persisting across sessions.
- User Profile: Display public profile information, including favorite tools, name, avatar and bio.

## Style Guidelines:

- Primary color: Deep slate blue (#2E3A59) to evoke a sense of stability and technical sophistication.
- Background color: Very dark gray (#12182B), almost black, maintaining the dark-mode-first approach.
- Accent color: Electric purple (#BE72F2), analogous to the primary, creating contrast and highlighting interactive elements.
- Headline font: 'Space Grotesk', sans-serif, for a techy, modern feel in large typographic layout. Body font: 'Inter', sans-serif, providing clean readability for descriptions and content.
- Use simple line icons to represent categories and interactive elements.
- Landing page layout featuring a large 'F orge H ive' typographic split. Logo centered with a horizontally aligned tech-revealing line.
- Subtle cursor-driven animations on the landing page, especially for the horizontal line. Smooth, minimal transitions throughout the app.
